# Engine Failure During Takeoff
1. ABORT. If engine failure occurs on takeoff before the
landing gear handle is raised and a barrier engagement/
arrestment is feasible, abort.

If barrier engagement/arrestment is not feasible or the
landing gear handle has been raised, proceed as follows:

1. EXTERNAL LOAD — JETTISON.
2. ZOOM IF POSSIBLE AND EJECT.

    !!! warning
        Any upward vector with wings level will
        increase changes of survival.
