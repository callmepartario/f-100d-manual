# Engine Failure During Takeoff
1. **ABORT**—If engine failure occurs on takeoff before the
landing gear handle is raised and a barrier engagement/
arrestment is feasible, abort.

If barrier engagement/arrestment is not feasible or the landing gear handle has been raised, proceed as follows:

1. External Load—**JETTISON**
2. **ZOOM**—If possible, then—**EJECT**

!!! warning
    Any upward vector with wings level increases chance of survival.
