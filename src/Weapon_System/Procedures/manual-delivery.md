# Manual Delivery

For specific delivery profiles see the [weapon release tables](../bomb-rockets.md)

## Rockets

The procedure below will setup the weapons so that the bomb-button will fire a salvo of rockets.

!!! note
    Rockets are most effective in ripple fire modes.

### 1. Rocket Arming

1. Armament Control Panel: [Stores Selector](../armament-control-panels.md#stores-selector)—**RKTS**
2. Armament Control Panel: [Mode Selector](../armament-control-panels.md#mode-selector)—**MANUAL**
3. Set All Armament Control Panel: [Station Release Selection Switches](../armament-control-panels.md#station-release-selection-switches)—**NORM**

### 2. Aircraft Weapons Release System (AWRS) Setup

1. Set AWRS [Quantity Selector](../aircraft-weapons-release-system.md#controls)—**AS DESIRED**
2. Set AWRS [Release Interval](../aircraft-weapons-release-system.md#controls)—**AS DESIRED**—(50 ms recommended)
3. Set AWRS [Mode Selector](../aircraft-weapons-release-system.md#controls)—**RIPPLE SALVO**


### 3. Sight Setup

1. Set [Sight Selector Switch](../a-4-gunsight.md#1-sight-selector-switch)—**ROCKET**
2. Set [Sight Depression](../a-4-gunsight.md#2-sight-depression)—**AS DESIRED**—(10-20 mils recommended)
3. Set [Mechanical Cage Lever](../a-4-gunsight.md#mechanical-cage-lever)—**UNCAGE**

## Bombs

The procedure below will setup the weapons so that the bomb-button will drop a series of bombs.

!!! note
    Bombs are most effective in ripple fire modes.

### 1. Bomb Arming

1. Armament Control Panel: [Stores Selector](../armament-control-panels.md#stores-selector)—**BOMBS**
2. Armament Control Panel: [Mode Selector](../armament-control-panels.md#mode-selector)—**MANUAL**
3. Set All Armament Control Panel: [Station Release Selection Switches](../armament-control-panels.md#station-release-selection-switches)—**NORM**

4. Set All Armament Control Panel: [Bomb Arm Switch](../armament-control-panels.md#bomb-arm-switch)—**NOSE & TAIL**

!!! note
    If dropping high-drag in low drag mode, set the [Bomb Arm Switch](../armament-control-panels.md#bomb-arm-switch) to **NOSE** instead.

### 2. Aircraft Weapons Release System (AWRS) Setup

1. Set AWRS [Quantity Selector](../aircraft-weapons-release-system.md#controls)—**AS DESIRED**
2. Set AWRS [Release Interval](../aircraft-weapons-release-system.md#controls)—**AS DESIRED**—(150 ms recommended)
3. Set AWRS [Mode Selector](../aircraft-weapons-release-system.md#controls)—**RIPPLE SINGLE**

### 3. Sight Setup

1. Set [Sight Selector Switch](../a-4-gunsight.md#1-sight-selector-switch)—**BOMB**
2. Set [Sight Depression](../a-4-gunsight.md#2-sight-depression)—**AS REQUIRED**—(See the [weapon release tables](../bomb-rockets.md))
3. Set [Mechanical Cage Lever](../a-4-gunsight.md#mechanical-cage-lever)—**UNCAGE**
